Learning Motion-Aware 4D Representations from Monocular Videos

Given a short input sequence, our anonymous system reconstructs temporally coherent 4D views and plays each result forward, pauses briefly, then reverses for direct inspection.

Abstract

Recent video-to-4D methods have made progress in reconstructing dynamic objects from monocular videos, but producing a usable animated mesh asset remains challenging. Frame-wise reconstruction methods can capture temporal shape variations, yet they often suffer from inconsistent topology and unstable correspondences. Reference-based methods improve temporal consistency by deforming a fixed mesh through time, but their results become highly dependent on the reference mesh. If the reference is reconstructed from a single frame, occlusion, missing geometry, or transient contacts in that frame can be inherited by the full sequence and cause distorted deformation in later frames. Multi-frame mesh consolidation can reduce this single-frame bias, but directly aligning detailed meshes from different frames often suppresses high-frequency geometry and appearance due to conflicting local reconstructions. We introduce a 4D-structure-guided pipeline for topology-consistent animated mesh reconstruction. Given an input video, we first predict a temporally consistent 4D coarse structure with a native 4D Sparse Structure model, and convert it into a canonical sparse structure that captures geometry compatible with the full motion sequence. We then regenerate a canonical SLAT from this structure to obtain a high-quality reference mesh and material representation using the native 3D prior. To preserve faithful appearance, we further bake reliable video observations onto the reference mesh, refine the lifted material latent through observation-preserved SDEdit, and enhance the decoded PBR maps with view-consistent image-prior guidance for sharper texture details. Finally, the reference mesh is animated by predicting per-frame vertex trajectories while keeping the mesh connectivity fixed. Our method produces cleaner reference geometry, sharper canonical materials, and more stable topology-consistent animation under large motion, changing visibility, and different object parts.
